// what this project covers
⚡
Automated CI/CD
Every push to GitHub triggers Jenkins pipeline — build, test, and deploy without manual steps.
🛡️
Code Quality Gates
SonarQube scans every build for bugs, vulnerabilities, and code smells before deployment.
🐳
Containerized App
Application packaged as a Docker image for consistent environments from dev to production.
☸️
K8s Deployment
Kubernetes manages scaling, rolling updates, and self-healing of the deployed containers.
📊
Live Monitoring
Prometheus collects metrics, Grafana visualizes them — CPU, memory, and request rates in real time.
🔁
Full Loop
Covers the complete DevOps lifecycle: plan → code → build → test → deploy → monitor.
